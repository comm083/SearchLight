# 🔦 SearchLight - YOLO 기반 CCTV 지능형 이벤트 분석 및 자연어 검색 시스템

> **마지막 업데이트: 2026-05-07**

CCTV 영상의 시각적 정보를 YOLO 객체 인식 모델로 분석하고, 이를 AI를 통해 요약하여 자연어로 쉽게 검색할 수 있는 지능형 보안 플랫폼입니다.
사용자가 "어제 편의점에서 무슨 일 있었어?"라고 입력하면, 시스템이 YOLO 분석 데이터와 RAG(검색 증강 생성) 기술을 결합해 상황을 요약하고 해당 영상 구간을 찾아줍니다.

---

## 🎯 프로젝트 목표

**YOLO 비전 AI와 RAG(자연어 처리) 기술**을 결합하여, 방대한 CCTV 이벤트 데이터를 실시간으로 모니터링하고 대화형으로 검색/분석할 수 있는 **지능형 보안 보조 시스템** 구축.

### 📌 해결하려는 문제

| **인간적 오류 (Human Error)**
장시간 CCTV 모니터링 시 중요한 장면을 놓치거나 잘못 해석하는 관제 오류가 발생할 수 있습니다.
YOLO가 이상 상황을 자동 감지하여 이 오류를 줄입니다.

| **장시간의 CCTV 기록 탐색**
긴 영상 기록에서 특정 사건이 발생한 시점을 수동으로 찾는 데 엄청난 시간이 소요됩니다.
자연어 검색으로 이를 해결 합니다.

| **비직관적인 인터페이스**
기존 시스템은 전문가용으로 복잡합니다. 일반 사용자도 챗봇 형식으로 쉽게 물어보고 답변을 얻을 수 있습니다.

### 🔧 핵심 기술 스택

### 👁️ 비전 (Vision AI)
- **YOLO-World, YOLOv8**: CCTV 영상 내 사람, 화재, 무단 침입 등 주요 객체 및 이상 상황 감지
- **CLIP & OpenCV**: 감지된 객체 영역 크롭 및 벡터 임베딩 추출, 타임스탬프 처리

### 🧠 자연어 및 검색 엔진 (NLP & Search)
- **KoELECTRA & SBERT**: 사용자 질의 의도 분류(5종) 및 텍스트 데이터 벡터화
- **FAISS / pgvector**: 비전/NLP 모델에서 추출된 벡터 간의 초고속 유사도 검색

### ⚙️ 시스템 및 인프라 (System & Infra)
- **FastAPI**: 파이썬 기반 고성능 백엔드 API 구축
- **React**: 사용자 친화적인 대화형 챗봇 UI/UX 
- **Supabase**: 검색 로그 및 이상 이벤트 데이터 실시간 저장 및 연동

## 👥 팀 역할 분담 (2인 체제)

본 프로젝트는 2인 체제로 운영되며, 비전 파이프라인과 NLP/데이터 파이프라인을 긴밀하게 통합합니다.

### 👤 김도윤 (AI & Data Pipeline)
- **NLP 모듈 고도화 및 환각 제어**: 자연어 의도 분류 및 시간 파싱 엔진 개선
- **RAG 및 검색 최적화**: 요약 프롬프트 엔지니어링 및 FAISS 기반 벡터 검색 고도화
- **비전 데이터 연동**: YOLO 탐지 객체 크롭(Crop) 이미지 추출 및 CLIP 모델 벡터화

### 👤 이건영 (Vision & Backend System)
- **YOLO 영상 분석 파이프라인 구축**: YOLO-World/v8을 활용한 CCTV 프레임 분석
- **객체 추적 및 시각화**: 이상 행동 탐지, 객체 추적 및 화면 내 바운딩 박스 오버레이 처리
- **시스템 통합**: FastAPI 백엔드 구축 및 Supabase 클라우드 DB 실시간 연동

## 🚀 주요 구현 기능 및 진행 현황 (Roadmap)

> **범례**: ✅ 완료 | 🔄 진행중 | ⬜ 미착수

### 👤 김도윤 : AI & Data Pipeline (환각 제어 및 검색 고도화)

- [ ] **1. 크롭 이미지(Crop Image) 추출 및 벡터화**
  - YOLO가 감지한 핵심 객체 영역만 크롭하여 저장하고, 이를 CLIP 등의 비전 모델로 임베딩하여 유사 인물/차량 검색의 정확도를 높입니다.
- [ ] **2. 최종 프로젝트 보고서 작성**
  - 내일 진행 예정인 최종 프로젝트와 관련된 종합 결과 보고서를 작성하고 문서화합니다.
- [ ] **3. 정밀 포즈 분석 (Pose Estimation) 연동**
  - MediaPipe 또는 YOLO Pose 모델을 도입하여 사람의 관절 위치를 파악하고, "팔을 휘두르는 동작" 등을 수치화하여 폭행/파손 등의 행동 분석 정밀도를 높입니다.

### 👤 이건영 : Vision & Backend System (YOLO 파이프라인 및 서비스 연동)

- [ ] **5. 엣지 디바이스(Edge Device) 배포 고려**
  - 서버 부하를 줄이기 위해 YOLO 추론을 카메라 단말이나 엣지 컴퓨팅(Jetson Nano 등)에서 처리하는 아키텍처를 검토합니다.
- [ ] **6. 실시간 서비스 운영 (추가)**
  - 영상이 녹화되는 동안에도 실시간으로 이상 행동을 분석하고 큰 문제가 발생 시 알림을 제공합니다.
- [ ] **7. 고밀도 버스트 샘플링 (Burst Sampling) 고도화**
  - 사건 발생 감지 시 해당 구간의 프레임을 초당 10장 이상 촘촘하게 추출하여, 찰나의 동작(예: 도구 꺼내기)을 놓치지 않도록 파이프라인을 개선합니다.
- [ ] **8. 실시간 모바일 푸시 알림 시스템**
  - 이상 상황 감지 시 관리자의 스마트폰으로 즉각적인 알림을 전송하는 실시간 연동 기능을 구축합니다.

## ⚡ 빠른 시작 (Quick Start)

### 1. 환경 설정 및 서버 구동

```bash
# 가상환경 활성화 (Windows)
.\the_searchlight\Scripts\Activate.ps1

# 백엔드 서버 실행 (API: http://127.0.0.1:8000)
.\the_searchlight\Scripts\python.exe -m uvicorn --app-dir backend app.main:app --reload

# 프론트엔드 서버 실행 (UI: http://localhost:5173)
cd frontend
npm run dev
```

### 2. 영상 분석 파이프라인 구동 (데이터 파싱)

YOLO 기반으로 새로운 영상을 분석하고 DB(혹은 JSON)로 변환할 때 사용합니다.

```bash
# 백엔드 디렉토리로 이동
cd backend

# mp4Data 폴더 내의 영상들을 읽어 YOLO 분석 진행
python makeData/makeJsonData.py --dir static/mp4Data
```
