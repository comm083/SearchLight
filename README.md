# 🔦 SearchLight - YOLO 기반 CCTV 지능형 이벤트 분석 및 자연어 검색 시스템

> **마지막 업데이트: 2026-05-07**

CCTV 영상의 시각적 정보를 YOLO 객체 인식 모델로 분석하고, 이를 AI를 통해 요약하여 자연어로 쉽게 검색할 수 있는 지능형 보안 플랫폼입니다.
사용자가 "어제 편의점에서 무슨 일 있었어?"라고 입력하면, 시스템이 YOLO 분석 데이터와 RAG(검색 증강 생성) 기술을 결합해 상황을 요약하고 해당 영상 구간을 찾아줍니다.

---

## 🎯 프로젝트 목표

**YOLO 비전 AI와 RAG(자연어 처리) 기술**을 결합하여, 방대한 CCTV 이벤트 데이터를 실시간으로 모니터링하고 대화형으로 검색/분석할 수 있는 **지능형 보안 보조 시스템** 구축.

### 📌 해결하려는 문제

| **인간적 오류 (Human Error)**
장시간 CCTV 모니터링 시 중요한 장면을 놓치거나 잘못 해석하는 관제 오류가 발생할 수 있습니다.
YOLO가 이상 상황을 자동 감지하여 이 오류를 줄입니다.

| **장시간의 CCTV 기록 탐색**
긴 영상 기록에서 특정 사건이 발생한 시점을 수동으로 찾는 데 엄청난 시간이 소요됩니다.
자연어 검색으로 이를 해결 합니다.

| **비직관적인 인터페이스**
기존 시스템은 전문가용으로 복잡합니다. 일반 사용자도 챗봇 형식으로 쉽게 물어보고 답변을 얻을 수 있습니다.

### 🔧 핵심 기술 스택

| 분야                   | 기술                         | 역할                                                                   |
| :--------------------- | :--------------------------- | :--------------------------------------------------------------------- |
| **비전 (Vision AI)**   | **YOLO-World, YOLOv8**       | CCTV 영상 내 사람, 화재, 무단 침입 등 주요 객체 및 이상 상황 감지      |
| **비전 (Vision AI)**   | **CLIP & OpenCV**            | 감지된 객체 영역 크롭 및 벡터 임베딩 추출, 타임스탬프 처리             |
| **자연어 (NLP / RAG)** | **KoELECTRA & SBERT**        | 사용자 질의 의도 분류(5종) 및 텍스트 데이터 벡터화                     |
| **검색 엔진**          | **FAISS / pgvector**         | 비전 모델과 NLP 모델에서 추출된 의미/특징 벡터 간의 초고속 유사도 검색 |
| **시스템 & 인프라**    | **FastAPI, React, Supabase** | 백엔드 API, 챗봇 UI, 클라우드 DB 연동                                  |

## 🚀 주요 구현 기능 및 진행 현황 (Roadmap)

> **범례**: ✅ 완료 | 🔄 진행중 | ⬜ 미착수

### 👤 김도윤 : AI & Data Pipeline (환각 제어 및 검색 고도화)

- [ ] **1. 크롭 이미지(Crop Image) 추출 및 벡터화**
  - YOLO가 감지한 핵심 객체 영역만 크롭하여 저장하고, 이를 CLIP 등의 비전 모델로 임베딩하여 유사 인물/차량 검색의 정확도를 높입니다.
- [x] **2. 채팅 타임라인 불일치 수정**
  - 채팅 내 표시되는 이벤트 타임라인과 실제 이벤트 타임라인이 다르게 출력되는 버그를 수정합니다.
- [x] **3. 상대적 시간 및 의도 분류 정확도 개선**
  - "3일 전", "N일 전" 등 상대적 시간 표현에 대한 검색 정확도를 높이고, 요약 질문이 수량 파악(COUNTING) 등 다른 의도로 분류되지 않도록 로직을 고도화합니다.
- [x] **4. 영상 데이터 타임스탬프 오버레이 자동화**
  - 신규 편의점 영상 데이터(18개) 확보 시, 각 영상의 좌측 상단에 랜덤하게 생성된 타임스탬프를 시각적으로 표시(Burn-in)하여 데이터의 시간적 일관성을 확보합니다.

### 👤 이건영 : Vision & Backend System (YOLO 파이프라인 및 서비스 연동)

- [x] **1. 녹화 영상 YOLO 분석 파이프라인 구축**
  - 저장된 영상 파일을 프레임 단위로 읽어 YOLOv8 (또는 YOLO-World) 추론을 수행하고, 분석 결과를 DB/JSON에 저장하는 프로세스를 구축합니다. (`makeJsonData.py` 등)
- [x] **2. YOLO 감지 이벤트 DB 실시간 적재**
  - 실제 YOLO가 감지한 객체(사람 등)와 이벤트를 Supabase `cctv_videos` 등 테이블에 Insert 합니다.
- [x] **3. 바운딩 박스(Bounding Box) 데이터 시각화 연동**
  - 프론트엔드에서 검색된 영상을 재생할 때, 필요한 경우 감지된 객체 영역을 식별할 수 있도록 파이프라인을 고도화합니다.
- [x] **4. 영상 타임스탬프와 YOLO 추론 시간 정밀 동기화**
  - 녹화 영상의 타임스탬프를 OCR로 추출하거나 동기화하여, 사용자가 특정 시간대를 검색했을 때 정확히 일치하는 영상이 반환되도록 합니다.
- [ ] **5. 엣지 디바이스(Edge Device) 배포 고려**
  - 서버 부하를 줄이기 위해 YOLO 추론을 카메라 단말이나 엣지 컴퓨팅(Jetson Nano 등)에서 처리하는 아키텍처를 검토합니다.
- [ ] **6. 실시간 서비스 운영 (추가)**
  - 영상이 녹화되는 동안에도 실시간으로 이상 행동을 분석하고 큰 문제가 발생 시 알림을 제공합니다.

## ⚡ 빠른 시작 (Quick Start)

### 1. 환경 설정 및 서버 구동

```bash
# 가상환경 활성화 (Windows)
.\the_searchlight\Scripts\Activate.ps1

# 백엔드 서버 실행 (API: http://127.0.0.1:8000)
.\the_searchlight\Scripts\python.exe -m uvicorn --app-dir backend app.main:app --reload

# 프론트엔드 서버 실행 (UI: http://localhost:5173)
cd frontend
npm run dev
```

### 2. 영상 분석 파이프라인 구동 (데이터 파싱)

YOLO 기반으로 새로운 영상을 분석하고 DB(혹은 JSON)로 변환할 때 사용합니다.

```bash
# 백엔드 디렉토리로 이동
cd backend

# mp4Data 폴더 내의 영상들을 읽어 YOLO 분석 진행
python makeData/makeJsonData.py --dir static/mp4Data
```
