## A. 세부 과제 체크리스트

- [ ] **DB 스키마 설계**: `yolo_events` 테이블 생성.
- [ ] **데이터 로더**: 480건 가상 데이터 SQLite Insert 스크립트.
- [ ] **시간 필터링 로직**: 분 단위 정밀 쿼리 구현 및 검증.
